# UI Designer

## Role Overview

The UI Designer is responsible for the visual design of the JTLD Consulting Inc platform, creating a professional, modern, and consistent look and feel that builds trust with both candidates and employers.

## Priority

**High** - Phase 1

## Core Technologies

| Technology | Purpose |
|------------|---------|
| Figma | Primary design tool |
| Adobe Photoshop | Image editing |
| Adobe Illustrator | Vector graphics and icons |
| Tailwind CSS | Design system integration |
| Shadcn UI | Component library reference |
| Storybook | Component documentation |

## Required Skills

### Visual Design Fundamentals

- Color theory and color palette creation
- Typography selection and hierarchy
- Layout composition and grid systems
- Spacing and alignment systems (8px grid)
- Visual hierarchy and emphasis
- Contrast and readability
- White space usage
- Consistency and pattern recognition

### Design Systems

- Component-based design methodology
- Design tokens (colors, spacing, fonts, shadows)
- Reusable component library creation
- Variant and state documentation
- Style guide maintenance
- Brand guidelines creation and enforcement
- Icon system design
- Illustration style guide

### Web-Specific Design

- Responsive design for multiple screen sizes
- Mobile-first design approach
- Touch-friendly interactive elements
- Web typography best practices (font loading, fallbacks)
- Image optimization for web
- Dark mode design
- Micro-interactions and animations
- Loading state design (skeletons, spinners)

### Component Design

- Buttons (primary, secondary, outline, ghost, sizes)
- Form inputs (text, select, checkbox, radio, date picker)
- Cards (job cards, profile cards, stat cards)
- Navigation (header, sidebar, breadcrumbs, tabs)
- Modals and dialogs
- Tables and data grids
- Alerts and notifications (toast, banner, inline)
- Badges and tags (skill tags, status badges)
- Avatars and user profiles
- Search bars and filter panels
- Pagination components
- Empty states and error states

### Page Design

- Landing/home page
- Job listing page with search results
- Job detail page
- User registration and login pages
- Candidate profile/dashboard
- Employer dashboard
- Job posting form
- Application review page
- Admin panel pages
- Settings and account pages
- 404 and error pages

### Color Palette for JTLD Consulting Inc

```
Primary:     #2563EB (Blue - trust, professionalism)
Secondary:   #0F172A (Dark Navy - authority)
Accent:      #10B981 (Green - success, growth)
Warning:     #F59E0B (Amber - attention)
Error:       #EF4444 (Red - errors, alerts)
Background:  #F8FAFC (Light gray)
Surface:     #FFFFFF (White)
Text:        #1E293B (Dark slate)
Muted:       #64748B (Gray - secondary text)
```

### Typography

```
Headings:    Inter or Plus Jakarta Sans (modern, clean)
Body:        Inter or system-ui (readable, performant)
Monospace:   JetBrains Mono (code, technical content)

Scale:
  h1: 36px / 2.25rem (page titles)
  h2: 30px / 1.875rem (section titles)
  h3: 24px / 1.5rem (card titles)
  h4: 20px / 1.25rem (subsection titles)
  body: 16px / 1rem (paragraph text)
  small: 14px / 0.875rem (captions, labels)
  xs: 12px / 0.75rem (badges, tags)
```

3. Create responsive layouts for all screen sizes
4. Design reusable UI components
5. Create design tokens for Tailwind CSS integration
6. Design micro-interactions and animations
7. Ensure visual consistency across all pages
8. Create dark mode variant designs
9. Design email templates
10. Provide design assets and specifications to developers
